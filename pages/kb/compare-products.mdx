# Compare products documentation

[Compare Products for WooCommerce](https://addonify.com/product/addonify-compare-products-for-woocommerce/) is a powerful plugin that compares two or more products on your woocommerce store. It provides a detailed comparison table that shows the key features and specifications of the products being compared.

This is a documentation section for the [Compare Products For WooCommerce](https://addonify.com/product/addonify-compare-products-for-woocommerce/) plugin. Here you will find all the necessary information to get started with the plugin along with the developer documentation.
