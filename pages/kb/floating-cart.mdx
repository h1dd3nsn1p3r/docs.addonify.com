# Floating cart documentation

[Floating Cart For WooCommerce](https://addonify.com/product/addonify-floating-cart-for-woocommerce/) is a powerful plugin that adds a side-cart functionality to your WooCommerce store.It allows users to quickly view and add items to the cart without having to navigate away from the page they are currently on.

This is a documentation section for the [Floating Cart For WooCommerce](https://addonify.com/product/addonify-floating-cart-for-woocommerce/) plugin. Here you will find all the necessary information to get started with the plugin along with the developer documentation.

Explore the topics below to get started.
