# Addonify wishlist docs

[Addonify Wishlist For WooCommerce](https://addonify.com/product/addonify-wishlist-for-woocommerce/) is a powerful plugin that allows users to add products to their wishlist directly from the product page. It helps to increase the engagement of your WooCommerce store and improve the user experience.

This is a documentation section for the [Addonify Wishlist](https://addonify.com/product/addonify-wishlist-for-woocommerce/). Here you will find all the necessary information to get started with the plugin along with the developer documentation. 

Explore the topics below to get started.
