# Support

We encourage you to browse [documentation guide](/kb) before reaching out to us. If the documentation doesn't help you, feel free to reach out to us. We are here to help you.

## Pre-sales questions

If you have any pre-sales related questions, write us at [contact@creamcode.org](mailto:contact@creamcode.org). We will get back to you as soon as possible.

We might also need following information to help you better:

- WordPress debug system information.
- Brief description of the issue along with screenshots.

Here's how you can retrieve the WordPress debug system information.

[![WordPress debug information](/support/debug-info.png)](/support/debug-info.png)

- Go to Dashboard > Tools > Site Health > Info tab.
- Click on the Copy system info to clipboard button.
- Create a text file and paste the copied system info.
- Save the file as `system-info.txt`.

While sending the email, please attach the `system-info.txt` file as attachment so that we can help you better. Here's the sample of email template:

```md
Subject: Hi, I need help with Addonify Quick View plugin

Email body:
Hi support,

I am facing an issue with the Addonify Quick View plugin. Here is the brief description of the issue:

..........................
<Describe the issue and if possible add screenshots>
..........................

I have attached the system-info.txt file with this email.

Thanks,
<Your name>
```
