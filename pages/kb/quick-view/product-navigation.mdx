# Quick view product navigation

Add quick product navigation to your Quick View popup modal. This feature allows customers to view the details of the next and previous products without closing the Quick View popup.


##### Enable products navigation ( pro )

Enable or disable the product navigation feature.


##### Display product image( pro )

Displays product image on navigation button.


[![Product meta](/products/quick-view/navigation-image.png)](/products/quick-view/navigation-image.png)




##### Display product title ( pro )

Displays product title on navigation button.


[![Product meta](/products/quick-view/navigation-product-title.png)](/products/quick-view/navigation-product-title.png)


##### Background color ( pro )

Sets the background color for the navigation buttons.


[![Product meta](/products/quick-view/navigation-bg-color.png)](/products/quick-view/navigation-bg-color.png)



##### Mx product title length ( pro )

If product title is too long then it trims the character as needed.

[![Product meta](/products/quick-view/max-title-length.png)](/products/quick-view/max-title-length.png.png)
