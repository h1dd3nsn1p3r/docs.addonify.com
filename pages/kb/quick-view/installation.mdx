# Installing Addonify quick view

You can install the Addonify WooCommerce Quick View plugin via the WordPress admin dashboard. The First process is simple & most commonly used.

### Process 1: Installing from the wordpress.org repository

1. Go to your WordPress admin dashboard.
2. Navigate to the `Plugins` menu.
3. Click on the `Add New` button.
4. Search for `Addonify Quick View` in the search bar.
5. Click on the `Install Now` button.
6. Once installed, click on the `Activate` button.
7. Done! You have successfully installed the Addonify Quick View plugin.

### Process 2: Installing from the zip file

1. Download the Addonify Quick View plugin from the [Addonify website](https://wordpress.org/plugins/addonify-quick-view/).
2. Go to your WordPress admin dashboard.
3. Navigate to the `Plugins` menu.
4. Click on the `Add New` button.
5. Click on the `Upload Plugin` button.
6. Choose the downloaded zip file.
7. Click on the `Install Now` button.
8. Once installed, click on the `Activate` button.
9. Done! You have successfully installed the Addonify Quick View plugin.

### Process 3: Installing via FTP

1. Download the Addonify Quick View plugin from the [Addonify website](https://addonify.com).
2. Extract the downloaded zip file.
3. Connect to your server via FTP.
4. Navigate to the `wp-content/plugins` directory.
5. Upload the extracted folder to the `plugins` directory.
6. Go to your WordPress admin dashboard.
7. Navigate to the `Plugins` menu.
8. Click on the `Installed Plugins` link.
