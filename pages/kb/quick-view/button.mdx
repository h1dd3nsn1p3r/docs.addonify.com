# Quick view button

Add a quick view button to your product list page. This button will allow your customers to view product details without leaving the page. 

**Note:** Developer? You can customize the quick view button. Explore the [developer section](/kb/quick-view/developer/).

## Button options

##### Button position

Choose where to display the Quick View button..

##### Button label

Label text for the button to display.

##### Enable icon in quick view button

The icon will be displayed before or after the Quick View button's label text..

##### Quick view button icons

Choose an icon to display..

##### Quick view button icon position

Choose the icon position to display: before or after the Quick View button's label text.

## Styles options

##### Text color

Quick view button label text color.

##### Background color

Background color for quick view button.

##### Border color

Border color for quick view button.

##### Border Style

Border style for quick view button.

##### Border width

Border width/thickness for quick view button.

##### Border radius

Border radius for quick view button.
