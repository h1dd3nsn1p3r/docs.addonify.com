# Quick view general options


##### Enable quick view

Enable or disable quick view feature.

##### Disable on mobile devices

Enable or disable quick view on moblie device.

##### Hide quick view button for product categories ( pro )

The `Quick View` button will be hidden for selected product categories.

##### Enable AJAX add to cart ( pro )

AJAX will be enabled for the `Add to Cart` button inside the modal box.

##### Action after AJAX add to cart ( pro )

Choose the action to take after the `Add to Cart` button is triggered.

##### Enable products view count ( pro )

Enables products view count for all products.
