# Addonify quick view docs

[Addonify Quick View For WooCommerce](https://addonify.com/product/addonify-quick-view-for-woocommerce/) is a powerful plugin that allows users to view product details without leaving the current page. It helps to reduce the bounce rate and increase the conversion rate of your WooCommerce store.

This is a documentation section for the [Addonify Quick View plugin](https://addonify.com/product/addonify-quick-view-for-woocommerce/). Here you will find all the necessary information to get started with the plugin along with the developer documentation. 

Explore the topics below to get started.
