# Commparision table


Compare Products plugin provides a compare button after add to cart button that allows you to add the product to the compare list on floating dock. If tou added more than two product on floating docks and trigger the compare button, it will show the comparison table.



## Compare Table Fields

This feature allows users to choose the fields to be displayed in the comparison table.


## Products attributes to compare

This feature allows users to choose the attributes to be displayed in the comparison table. There are two attributes: color and size.

## Show Table Fields Header

By enabling this feature, the header of the comparison table will be displayed.
