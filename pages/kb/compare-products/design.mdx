# Compare Products Design Page

## Interface Design

### Enable dynamic styles

By enabling this feature, styles from plugin will be applied.


## Compare Button Colors

### Label Color

This feature allows you to change the color of the compare button label.


[![Compare button label](/products/compare-products/compare-button-label.png)](/products/compare-products/compare-button-label.png)


##  Floating Dock Colors

### Background Color

This feature allows you to change the background color of the floating dock.


[![Floating dock](/products/compare-products/floating-dock.png)](/products/compare-products/floating-dock.png)


## Search Modal Colors

### Overlay Color

This feature allows you to change the background color of the search modal overlay. When user press product add button to compare products, search modal will open.



### Add Button Label Color

This feature allows you to change the color of the add button label in the search modal.

[![Add button label](/products/compare-products/add-button-label.png)](/products/compare-products/add-button-label.png)


### Close Button Label Color

This feature allows you to change the color of the close button label in the search modal.

[![Close button label](/products/compare-products/close-button-label.png)](/products/compare-products/close-button-label.png)


## Comparison Table Colors
