# Settings


## General Options

### Enable Products Comparison

By enabling this feature, users can compare products and view the comparison table. If this feature is disabled, users will not be able to compare products.



### Enable Login Required

By enabling this feature, users must be logged in to compare products. If this feature is disabled, users can compare products without being logged in.

### Enable Products Comparison on Products Loop

By enabling this feature, users can compare products on the product loop page. If this feature is disabled, the compare product button will not be displayed on the product loop page.

### Products Comparison Display

This feature allows users to choose whether to display the comparison table on popup modal or on the product detail page.

### Products Comparison Page

This feature allows users to view the comparison table on selected page.

### Time to Save Compare Data

This feature allows users to choose the time to save the compare data in browser cookies. Default value is 30 days.


## Compare Button Options

### Button Position in Products Loop                 

Choose where to display the compare button. There are two options: Before Add to Cart Button and After Add to Cart.

### Show Icon

By enabling this feature, the icon will be displayed before or after the compare button's label text.

### Icon Position

Choose where to display the compare button icon. There are two options: Before button label and After button label.

## Comparision table

### Compare Table Fields

This feature allows users to choose the fields to be displayed in the comparison table.


### Products attributes to compare

This feature allows users to choose the attributes to be displayed in the comparison table. There are two attributes: color and size.

### Show Table Fields Header

By enabling this feature, the header of the comparison table will be displayed.
