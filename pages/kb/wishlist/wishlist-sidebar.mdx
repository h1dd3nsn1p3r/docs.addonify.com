# Wishlist Sidebar

## Sidebar Options

### Display Sidebar

Enable this feature to display the wishlist sidebar on the shop page.

[![display sidebar](/products/wishlist/wishlist-sidebar.png)](/products/wishlist/wishlist-sidebar.png)

### Sidebar position

Choose where to display the wishlist sidebar. There are two options: Left and Right.

### Sidebar title

This feature allows you to change the title of the wishlist sidebar.


### Wishlist page button link label

This feature allows you to customize the label of the wishlist page button link button in the wishlist sidebar.

[![Wishlist page button link label](/products/wishlist/wishlist-link-button.png)](/products/wishlist/wishlist-link-button.png)


## Sidebar Toggle Button Options

### Button label

This feature allows you to change the label of the toggle button in the wishlist sidebar.


[![wishlist link button label](/products/wishlist/sidebar-toggle-button-label.png)](/products/wishlist/sidebar-toggle-button-label.png)


### Buton posiotion offset

This feature allows you to set the left and right offset of the toggle button in the wishlist sidebar.



## Sidebar Product Color Options

###  Product name color

This feature allows you to change the color of the product name in the wishlist sidebar.
