# Installing Addonify quick view

You can install the Addonify WooCommerce Wishlist plugin via the WordPress admin dashboard. The First process is simple & most commonly used.

## Installing from dashboard

1. Go to your WordPress admin dashboard.
2. Navigate to the `Plugins` menu.
3. Click on the `Add New` button.
4. Search for `Addonify Wishlist` in the search bar.
5. Click on the `Install Now` button.
6. Once installed, click on the `Activate` button.
7. Done! You have successfully installed the Addonify Quick View plugin.


## Installing from the zip file

1. Download the Addonify Wishlist plugin from the [wordpress.org website](https://wordpress.org/plugins/addonify-wishlist/).
2. Go to your WordPress admin dashboard.
3. Navigate to the `Plugins` menu.
4. Click on the `Add New` button.
5. Click on the `Upload Plugin` button.
6. Choose the downloaded zip file.
7. Click on the `Install Now` button.
8. Once installed, click on the `Activate` button.
9. Done! You have successfully installed the Addonify wishlist plugin.

## Installing via FTP

1. Download the Addonify Quick View plugin from the [wordpress.org website](https://addonify.com).
2. Extract the downloaded zip file.
3. Connect to your server via FTP.
4. Navigate to the `wp-content/plugins` directory.
5. Upload the extracted folder to the `plugins` directory.
6. Go to your WordPress admin dashboard.
7. Navigate to the `Plugins` menu.
8. Click on the `Installed Plugins` link.

## Installing the pro version

In-order to install the [Addonify Wishlist Pro](https://addonify.com/product/addonify-wishlist-for-woocommerce/) plugin, you need to have the free version installed on your WordPress site. Once you have the free version installed, you can install the pro version by following the below steps.

1. Download the Addonify Wishlist Pro plugin's zip file from the [CreamCode dashboard](https://creamcode.org/login).
2. If you are in the apple devices (mac), based on the download preference the zip file will be extracted automatically. Create a zip file of the extracted folder.
