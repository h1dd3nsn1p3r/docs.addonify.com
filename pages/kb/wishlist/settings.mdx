# Wishlist settings page


Managing the settings & options of the Addonify Wishlist plugin is easy. Once you have installed the plugin, you will find a menu item in your WordPress Dashboard labelled “Addonify“. Click the label to access the setting page of the Addonify Wishlist plugin.

[![Access plugin settings page](/products/wishlist/access-plugin-setting-page.png)](/products/wishlist/access-plugin-setting-page.png)

Once you are inside the Addonify Wishlist plugin setting page, as shown in the above image you will find 8 sub-pages:

1. General Settings
2. Popup Modals
3. Wishlist button
4. Wishlist Page
5. Wishlist Sidebar
6. Wishlist Notice
7. tools
8. Products
