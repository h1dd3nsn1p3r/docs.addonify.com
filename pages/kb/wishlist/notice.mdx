# Wishlist notice options

Options related to wishlist notice.

## Notice timeout (in seconds)

This feature allows you to change the timeout of the wishlist notice. The default value is 5 seconds.


[![wishlist notice](/products/wishlist/wishlist-notice.png)](/products/wishlist/wishlist-notice.png)



## Product removal undo notice options


## Text before undo link

This feature allows you to customize the text before the undo link in the product removal undo notice.


## Undo link label

This feature allows you to change the label of the undo link in the product removal undo notice.
